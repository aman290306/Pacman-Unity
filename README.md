# Pacman Unity

A 2D recreation of the classic Pac-Man arcade game developed using Unity and C#.

## Features

* Player-controlled Pac-Man movement
* Pellet collection system
* Score tracking
* Ghost enemy AI
* Collision detection
* Win and Game Over conditions

## Technologies Used

* Unity 6
* C#
* Visual Studio

## How to Run

1. Clone the repository
2. Open the project using Unity Hub
3. Open the main scene
4. Press Play
